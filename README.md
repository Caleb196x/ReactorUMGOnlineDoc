# ReactorUMG 文档站

ReactorUMG 是基于 React/TypeScript 的虚幻引擎 UMG/UI 插件，本仓库存放其官方文档，使用 MkDocs Material 构建，涵盖概览、快速入门、示例、API 与 FAQ。

## 快速开始
- 依赖：Python 3.10+，`pip install -r requirements.txt`
- 本地预览：`mkdocs serve`，默认 http://127.0.0.1:8000
- 构建静态站点：`mkdocs build`，输出在 `site/`
- 部署网站：`mkdocs gh-deploy --clean`

## 文档结构
- `docs/index.md`：项目概览与特性
- `docs/quickstart.md`：搭建与运行步骤
- `docs/examples/`：界面与编辑器示例模版
- `docs/api/`：接口与组件占位
- `docs/en/`：英文文档入口
